NASA EGT LUT (N-CMAPSS DS02)

Download the N-CMAPSS DS02 dataset file: N-CMAPSS_DS02-006.h5

Link:  https://figshare.com/articles/dataset/N-CMAPSS_DS02-006_h5/20436504?file=36563133


Place the .h5 file in:
data/raw/

Run the LUT builder script:
scripts/build_egt_lut.m

This generates the processed LUT file:
data/processed/egt_lut_v1.mat
